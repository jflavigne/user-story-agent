# Reference Materials for Mockup Analysis Skill

This folder contains research and examples gathered for building a skill that:
1. Reviews website/desktop mockups
2. Analyzes and breaks them down into components and tasks
3. Plans what user stories are required
4. Gets user approval then writes the stories
